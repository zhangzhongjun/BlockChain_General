# truffle框架

## truffle.js

位于项目的根目录下，是项目的配置文件。

## 使用例子项目

> truffle框架提供了一些例子项目

```bash
mkdir MetaCoin
cd MetaCoin
truffle unbox metacoin
```

### 项目结构

* contracts/: Directory for Solidity contracts
